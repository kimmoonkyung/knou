## 요약2_리눅스 설치
```
  01. CentOS 리눅스는 레드햇 엔터프라이즈 리눅스의 소스코드를 사용하여 만들었다.
  
  02. CentOS 리눅스를 설치하기 위해 ISO 이미지 파일을 준비해야 한다.
  
  03. 리눅스를 설치하는 방법으로는 '하드디스크의 비어 있는 파티션에 설치'하는 방법과 '가상머신 소프트웨어를 이용하여 게스트 운영체제로 설치'하는 방법이 있다.
  
  04. 가상머신 소프트웨어란 하나의 물리적 컴퓨터에 추가로 가상머신을 만들어 주는 소프트웨어이다.
  
  05. 저장 장치를 사용하려면 해당 저장 장치 이름을 파일 시스템의 특정 디렉터리에 마운트시켜야 한다.
  
  06. 리눅스에는 하나의 루트 디렉터리(/)만 존재한다. 또 용도가 정해진 표준 디렉터리들이 존재한다.
  
  07. /etc 디렉토리에는 시스템의 중요한 환경 설정 파일과 'passwd' 파일, 'fstab'파일 등이 저장된다.
  
  08. 리눅스를 설치할 때 루트 파티션(/)과 swap 파티션이 반드시 필요하다.
  
```

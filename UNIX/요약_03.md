요약3_쉘사용하기
```
  01. 셸은 운영체제 커널과 사용자 사이에 존재하는 명령어 해석기 또는 명령 행 인터페이스이다.
  
  02. Bash 셸은 CentOS를 포함한 많은 리눅스 배포판에서 기본 셸로 사용된다.
  
  03. 셸 명령의 종류로는 에일리어스, 셸 예약어, 함수, 내장 명령, 일반 명령 등이 있다.
  
  04. 히스토리 기능과 명령 행 자동 완성 기능을 사용하면 빠르고 편리하게 명령 행을 작성할 수 있다.
  
  05. 파이프(|)는 왼쪽 명령의 결과를 오른쪽 명령의 입력으로 연결시킨다.
  
  06. 입출력 리다이렉션은 셸 명령의 입력 또는 출력을 파일로 연결시킨다.
  
  07. 명령 확장을 수행할 때 큰따옴표("")는 작은따옴표('')와 다르게 따옴표 안에서 문자 $, ₩, \를 해석한다.
  
  08. 셸을 시작할 때 자동으로 수행되는 스크립트 파일이 존재한다.
```

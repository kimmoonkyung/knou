## 요약8_웹과 멀티미디어
```
  01. 월드 와이드 웹, 줄여서 웹은 인터넷에 의해 서로 거미줄처럼 연결되어 있는 수많은 사이트와 데이터, 멀티미디어 정보들을 언제, 어디에서나 필요한 곳에 접근할 수 있고, 공유할 수 있또록 하는 정보시스템 또는
      정보공간을 말한다.
  
  02. 인터넷이란 TCP/IP라는 전송규약을 사용하여 연결된 모든 네트워크를 말하며, 인터넷에 연결되어 있는 모든 종류의 네트워크를 포함한다.
  
  03. 웹과 관련 된 기술은 월드 와이드 웹 컨소시엄(W3C)에서 개발되고 있으며, W3C(World wide Web Consortium)는 웹을 위한 표준을 개발하고 장려하는 조직이다.
  
  04. 인터넷상의 정보 교환을 위해 웹에서는 지리적으로 분산 된 텍스트 데이터들을 하이퍼링크를 사용하여 상호 연결되도록 하는 하이퍼텍스트 구조를 가진다.
  
  05. 웹 2.0이란 기존의 웹이 일방적으로 정보를 전달하는 것에 반해 인터넷 상에서 양방향으로 정보를 주고 받아 서로 공유하며, 참여하는 새로운 인터넷 환경을 말한다.
  
  06. 웹 3.0은 웹이 앞으로 어떻게 될 것인가를 정의할 때 쓰이는 용어이다. 웹의 기술적인 발전에 대한 구체적인 표현이 될 수도 있고, 웹의 진화에 대한 바람을 담고 있을 수도 있으며, 지금까지와는 전혀 다른
      새로운 응용 프로그램들을 의미하는 것일 수도 있다.
  
  07. 웹 페이지는 인터넷상의 웹 문서들을 나타내는 표현으로, 다양한 텍스트와 이미지, 사운드, 비디오 파일 등으로 구성되며, 다른 인터넷 문서와 서로 연결할 수 있게 해 주는 하이퍼링크 기능이 있다.
  
  08. HTML은 SGML 언어의 한 종류로 하이퍼텍스트 문서를 만들기 위하여 쓰이는 간단한 마크업 언어이다.
  
  09. HTML 이외의 웹 문서의 표준으로 SGML, XML, DHTML 등이 있다.
  
  10. 웹 페이지에 동적인 기능을 지원하는 기술에는 자바스크립트, DHTML, CGI, ASP, JSP, PHP 등이 있으며, 그 밖에 애니메이션 기능을 위주로 한 플래시, 가상현실을 지원하는 VRML 등이 있다.
  
```

## 연습문제
```
  1. 인터넷의 특징으로 옳지 않은 것은 ?
    1) 하이퍼텍스트 언어이다.
    2) HTML 명령어는 태그를 사용하여 삽입한다.
    3) 일반적인 텍스트 문서이다.
    4) HTML 문서의 작성은 일반적인 텍스트 에디터를 통해서 작성할 수 없다.       - 정답.
    
  2. 다음 XML에 관한 설명 중 옳지 않은 것은 ?
    1) 사용자가 내용에 관련 된 태그를 직접 만들 수 있다.
    2) 정해진 마크업 언어만 사용하므로 다양한 클래스로 문서를 표현하는 것이 불가능하다.      - 정답
    3) 장치와 시스템에 독립적이다.
    4) 기존에 사용하던 HTML의 한계를 극복하고 SGML의 복잡함을 해결하는 방안이다.
  
  3. 다음 중 웹 2.0의 특징을 설명하는 것에 해당하지 않는 것은 ?
    1) 개방과 참여
    2) 플랫폼으로서의 웹
    3) 가벼워진 웹 S/W
    4) 독점, 배타적 구조         - 정답.
  
  4. HTML 문서에서 사용되는 태그에 대한 설명으로 옳지 않은 것은 ?
    1) 일반적으로 태그는 <  > 으로 둘러싸서 구별한다.
    2) 대-소문자를 구별한다.                   - 정답
    3) HTML 문서에서 공백문자는 무시된다.
    4) 대부분의 태그들은 시작 태그와 종료 태그 형식을 가진다.
    
  5. 웹 페이지에 동적인 기능을 지원하는 기술로 적당하지 않은 것은 ?
    1) 자바스크립트
    2) CGI
    3) HTML       - 정답
    4) 플래시
```

## 요약4_사운드
```
  01. 사운드는 기본적으로 물리적인 진동 작용인 음파를 말하며 음악, 목소리, 단순한 소리 등을 말한다.
 
  02. 아날로그 사운드는 우리가 실제 세상에서 듣는 소리를 의미하며 음의 크기(진폭), 음의 높이(주파수), 음색(소리의 색깔)으로 구성되어 있다.
  
  03. 디지털 사운드는 사운드가 디지털로 표현되고 저장 된 것을 의미하며 컴퓨터는 오직 디지털 신호만을 처리할 수 있기 때문에 아날로그 신호를 디지털 신호로 변환하는 작업이 필요하다.
  
  04. 사운드의 변환이란 아날로그 신호를 디지털 신호로 바꾸는 것으로 샘플링, 양자화, 부호화 과정을 거치게 된다. 
      샘플링은 파형을 일정한 간격으로 잘게 쪼개어 각 지점의 값을 하나하나 차례대로 끄집어내는 작업이고,
      양자화는 어떤 값을 취할 수 있는 유한의 이산적인 값으로 표현하는 조작을 말한다.
      부호화는 양자화 된 아날로그 신호들을 디지털 형태의 2진 정보로 표현하는 것이다.
      
  05. PCM(Pulse Code Modulation) 방식은 입력 된 값을 그대로 저장하는 방법으로, 아날로그 파형을 작은 시간 간격으로 연속적으로 나누어 각기 직사각형 형태의 크기로 표시한 후 이의 높이를 숫자화 한다.
  
  06. 컴퓨터를 이용하여 음악 CD나 음향효과 CD의 사운드를 추출하거나 컴퓨터의 사운드 카드와 오디오시스템에 연결하여 CD, 테이프, 라디오 등에서 나오는 소리나 사람의 목소리를 녹음할 수도 있다.
  
  07. 대표적인 디지털 오디오 파일 형식에는 WAV(Waveform Audio Format), AU(AUdio), AIFF(Audio Interchange File Format), RealAudio, ASF 등이 있다.
  
  08. 디지털 사운드 편집 프로그램에는 사운드 포지와 골드웨이브 등이 있다.
  
  09. MIDI(Musical Instrument Digital Interface)는 디지털 음악에 대한 국제 표준으로 제조업체가 다른 컴퓨터와 전자악기를 연결하는 하드웨어와 케이블 및 장비 간의 통신 프로토콜에 관한 사양을
      통신규약으로 정한 것이다.
      
  10. 시퀀서(sequencer)는 멀티 트랙 레코더를 시뮬레이션한 프로그램으로 악기로 연주되는 음악을 트랙이라는 단위로 녹음했다가 녹음 된 여러 트랙들을 동시에 재생함으로써 마치 오케스트라를 연주하는 것과 같은
      효과를 내는 미디 편집 프로그램이다.
  
  11. 노테이션 프로그램은 직접 손으로 입력한 미디 신호를 변환 한 후 컴퓨터 그래픽을 이용하여 악보로 나타내는 프로그램이다.
  
  12. 미디 파일 형식에는 RMI, MOD, SMF 등이 있다.
  
```

## 연습문제
```
  1. 다음 중 아날로그 사운드를 디지털 데이터로 변환하기 위해 샘플링 된 신호의 크기를 이산적으로 단계화하는 과정은 ?
    1) 샘플링
    2) 양자화          - 정답
    3) 부호화
    4) 샘플링 비율
  
  2. 사운드의 변환이란 (    ) 신호를 (    ) 신호로 바꾸는 것을 말한다. (    ) 안에 들어갈 단어를 순서대로 나열한 것은 ?
    1) 아날로그, 디지털      - 정답
    2) 아날로그, 양자화
    3) 디지털, 아날로그
    4) 아날로그, 샘플링
    
  3. 오디오 파일의 형식이 아닌 것은 ?
    1) WAV
    2) AU
    3) RealAudio
    4) PCM            - 정답
  
  4. 오실로스코프를 통해 사운드를 실시간으로 진폭과 스펙트럼으로 나타낼 수 있고 대용량 파일의 융통성 있는 편집, 다양한 효과, 다양한 사운드 포맷 등을 지원하는 디지털 오디오 편집 소프트웨어는 ?
    1) 케이크워크
    2) 노테이션
    3) 골드웨이브        - 정답
    4) 음성 녹음기
    
  5. 미디 파일 형식인 MOD(MODules)에 대한 설명으로 틀린 것은 ? 4번
    1) WAV 같은 음원도 함께 내장되어 있다.
    2) 샘플링 된 소리 데이터들을 소프트웨어적인 방법으로 변형, 합성해 음악으로 출력한다.
    3) 아미가(AMIGA) 시스템에서 개발한 것이다.
    4) 표준 미디 파일이다.                                                - 정답
    
  6. (    )는(은) 악기로 연주되는 음악을 트랙이라는 단위로 녹음했다가 녹음된 여러 트랙들을 동시에 재생함으로써 마치 오케스트라를 연주하는 것과 같은 효과를 내는 프로그램이다. (    ) 안에 알맞은 말은 ?
    1) 시퀀서              - 정답
    2) 노테이션 프로그램
    3) MIDI
    4) 보이싱 
```
